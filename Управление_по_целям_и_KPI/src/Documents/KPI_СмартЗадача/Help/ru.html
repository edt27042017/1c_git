Документ "Смарт задача"

Документ предназначен для ввода в систему SMART - задач,  их весов, планируемого выполнения, критериев оценки и описания. Работа с документом осуществляется из обработки "управление по целям". Проводить документ может пользователь, имеющий на это права (настраивается в справочнике "Должности") или пользователь, с ролью "Полные права". Документ SMART - задача может быть введен только один на период планирования.

 Интерактивная работа с документом возможно только при наличии "полных прав". Пользователи с ролями, отличными от роли "Полные права" могут только просматривать документ, созданный автоматически. Открыть список документов можно из меню Документы -> Планирования -> СМАРТ-задачи.

В шапке документа указывается:

Объект управления - объект управления, для которого устаннавливаются планы (справочник "Структура компании").
