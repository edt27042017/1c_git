Документ "Ввод фактов выполнения"

Документ предназначен для ввода в систему фактов по целевым показателям за период планирования. Работа с документом осуществляется из обработки "управление по целям". Создавать и проводить документ может пользователь, имеющий на это права (настраивается в справочнике "Целевые показатели") или пользователь, с ролью "Полные права".

Документ может быть вызван из меню Документы -> Факты выполнения -> Ввод фактов выполнения.

В шапке документа указывается:

Объект управления - объект управления, для которого вводятся факты (спраовчник "Структура компании").

Сценарий - сценарий планирования (справочник "Сценарии планирования").

В табличной части указываются целевые показатели (справочник "Целевые показатели")  и фактическое значение их выполнения на дату документа.
