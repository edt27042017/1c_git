Документ "Планирование"

Документ предназначен для ввода в систему целевых показателей,  их весов и планируемого выполнения. Сумма весов целевых показателей должна быть равна 100%. Работа с документом осуществляется из обработки "управление по целям". Проводить документ может пользователь, имеющий на это права (настраивается в справочнике "Модель планирования") или пользователь, с ролью "Полные права". Если установлен флаг "Подписан", то элементы формы документа закрыты для редактирования.

Документ может быть вызван из меню Документы -> Планирование -> Ввод планов на период. Интерактивная работа с документом разрешена пользователям, с ролью "Полные права".

В шапке документа указывается:

Объект управления - объект управления, для которого устанавливаются планы (справочник "Структура компании").

Сценарий - сценарий планирования (справочник "Сценарии планирования").

Модель - модель планирования (справочник "Модель планирования").

Подписан - признак утверждения планов.
