Документ предназначен для ввода в систему стандартов,  их важности и оценок выполнения. Стандарты (справочник "Структура компании") – это оценка соблюдения объектом управления правил поведения, технологических, рабочих и должностных инструкций. Проводить документ может пользователь, имеющий на это права (настраивается в справочнике "Модель планирования") или пользователь, с ролью "Полные права". Документ может быть создан только один на заданный период планирования.
Работа с документом осуществляется из обработки "управление по целям". Интерактивная работа с документом возможно только при наличии "полных прав". Пользователи с ролями, отличными от роли "Полные права" могут только просматривать документ, созданный автоматически. Открыть список документов можно из меню Документы ->Планирование -> Ввод стандартов .
В шапке документа указывается:
Объект управления - объект управления, для которого устаннавливаются планы (справочник "Структура компании").
Сценарий - сценарий планирования (справочник "Сценарии планирования").
Модель планирования - модель планирования (справочник "Модель планирования").
В табличной части указываются стандарты, их важность, оценки выполнения и комментарии к оценкам, которые вводятся пользователем, оценивающим текущий объект управления.
