Справочник "Матрица распределения КПД"

Справочник предназначен для хранения информации о расчете суммарного КПД, на базе которого будет производится расчет премии объекту управления. Таким образом, премия сотруднику будет начисляться не только исходя из выполнения его личного плана, но и выполнения плана отделом, подразделением, компанией.

Открыть список элементов справочника можно из меню: Справочники  - > Настройка системы вознаграждений  -> Матрица распределения.

В табличной части указывается объекты управления, КПД которых будет участвовать в формировании суммарного КПД и их вес при его формировании. Сумма весов КПД объектов управления, заведенных в табличной части должна быть равна 100.
