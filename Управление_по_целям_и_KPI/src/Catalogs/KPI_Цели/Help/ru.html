Справочник "Цели"

В справочнике хранится перечень целей, по которым будет оцениваться объект управления.

Цель представлена как элемент справочника «Цели». При создании новой цели необходимо в шапке элемента указать перспективу, в которую входит данная цель. На вкладке показатели задается список показателей, входящих в цель, с отбором по аналитикам и структурным подразделениям. Суммарный вес всех показателей должен быть равен 100%, иначе система выдаст предупреждение и не даст записать.

На вкладке планируемые значения указывается планируемое итоговое значение по данной цели и отклонения от нее в процентах. Данные значения нужны для отображения индикаторов достижения цели в стратегической панели.

После создания списка всевозможных целей в справочнике, необходимо создать документ Взаимосвязи целей. Данный документ описывает структуру целей для выбранного структурного подразделения на период. Документ формирует движение в регистре сведений, соответственно при формировании стратегической панели берётся срез последних, поэтому если взаимосвязи целей не будут меняться длительное время, достаточно сделать 1 документ для каждого структурного подразделения.
