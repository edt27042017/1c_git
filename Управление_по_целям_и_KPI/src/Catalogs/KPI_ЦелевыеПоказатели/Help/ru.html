Справочник "Целевые показатели"

В справочнике «Целевые показатели» перечислены ключевые показатели деятельности всех сотрудников организации и подразделений. В разрезе целевых показателей происходит планирование и оценка результативности и рассчитывается итоговый коэффициент результативности.

    Описание реквизитов:

  • Тип расчета
    Тип расчета предназначен для указания способа ввода данных о факте выполнения целевого показателя. Способ расчета показателя Для анализа работы объекта управления в разрезе показателя, могут использоваться следующие схемы: «взвешенный» или «абсолютный». Например, такой показатель как «Объем валовой прибыли» будет иметь тип расчета «накопительный», т.е. при анализе выполнения плана по этому показателю за период в полгода при помесячном планировании факт выполнения будет суммироваться. А показатель «Удовлетворенность клиентов», единицей измерения которого является «процент», за период в полгода при помесячном планировании будет рассчитан как сумма факта, деленная на количество месяцев, т.е. тип расчета «Среднеарифметический».
  • Единицы измерения
    Целевой показатель характеризуется базовой единицей измерения. Если целевой показатель уже использовался при планировании, то изменить базовую единицу измерения невозможно. При заполнении классификатора единиц измерения, справочник заполняется в соответствии с общероссийским классификатором единиц измерения. При заполнении «Единиц измерения» используется «классификатор единиц измерения».
На вкладке «Описание» можно задать текстовое описание. на вкладке «Аналитика» можно задать список аналитических разрезов целевого показателя. Также при выборе «Тип расчета» , «Автоматически по формуле» (доступны только для показателя с динамическим типом расчета) появляются вкладки для указания формул, расчет значения по этому показателю в таком случае будет происходить в момент вывода данных, например, в тактической панели показателей) На вкладке «Свойства» указывается ряд параметров, связанных со сбором данных по этому показателю . При выборе типа расчета «Автоматически механизмом сбора фактов» становится доступной вкладка «Источник данных», «источник данных факт» здесь не заполняется, он заполняется после того, как в источнике для сбора будет указана ссылка на показатель (подробнее в главе 13). Предельный показатель - признак того, что показатель является предельным, т.е. при увеличении факта процент его выполнения снижается. Обратный показатель - когда коэффициент результативности считается как план, деленный на факт (если план 100, а факт по показателю 50, то коэффициент результативности равен 200). Так же можно выбрать агрегирование результата:
  • Сумма;
  • Минимум;
  • Максимум;
  • Среднее.

На вкладке «Свойства» есть подвкладка «Индикаторы». На ней можно задать интервалы тренда и зоны в процентном соотношении, для последующего вывода динамики в отчете «Показатели в разрезе аналитик». По вхождению значения показателя в заданные интервалы будет изменяться отображение его тренда и зоны в отчете. Если значение тренда (факт – факт.предыдущий / факт.предыдущий) меньше нижней границы тренда (нижняя и верхняя границы тренда задаются в процентах, от 0 до 100), то в отчете («показатели по аналитике») в качестве индикатора тренда отобразится , если между нижней и верхней границей, то , и если больше верхней соответственно . Аналогичным образом настраиваются и границы зоны.

Предопределенные элементы

Показатель «SMART-задачи» (Целевые показатели) консолидировано представляет информацию по выполнению всех поставленных задач сотрудником. Данные по показателю вносятся одноименным документом.
