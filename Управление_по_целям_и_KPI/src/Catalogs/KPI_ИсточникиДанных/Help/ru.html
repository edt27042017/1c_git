Справочник "Источники для выборки данных"
В справочнике хранятся источники для выборки данных, которые поставляют данные для автоматического расчета целевых показателей. В качестве источника может выступать регистр накопления или произвольный запрос. Произвольный запрос можно использовать только в текущей информационной базе. Регистр накопления можно использовать как в текущей базе, так и в режиме COM - соединения.
