В справочнике хранится перечень стандартов, по которым будет оцениваться объект управления и пользователи, имеющие право проставлять оценки по стандартам.

Оценки по стандартам проставляются по истечении периода планирования. Возможны следующие варианты оценок: неприемлемо для компании, ниже ожидаемого, соответствует ожидаемому,  выше ожидаемого, выдающееся поведение.

Права на проставление факта по стандартам - это пользователи имеющие право проставлять оценки по данному стандарту. Имеет составной тип данных: справочник "Структура компании"  или перечисление "Структура пользователей".
