Справочник "Виды контактной информации"
 

Справочник служит для классификации контактной информации различных объектов системы.

Для каждого вида контактной информации определен тип контактной информации из заложенного в системе фиксированного списка, и тип объекта, для которого данный вид контактной информации будет использоваться.

Некоторые элементы справочника «Виды контактной информации» являются предопределенными и не удаляются пользователями.
